## ğŸš€ Getting Started

### Local Development

1. **Clone or download** the website files
2. **Open terminal** in the project directory
3. **Start local server**:
   ```bash
   python -m http.server 8000
   ```
4. **Open browser** and go to `http://localhost:8000`

### Deployment Options

#### Option 1: Static Hosting (Recommended)
- **Netlify**: Drag and drop the folder to Netlify
- **Vercel**: Connect GitHub repo for automatic deployments
- **GitHub Pages**: Upload to GitHub and enable Pages

#### Option 2: Traditional Hosting
- Upload all files via FTP to your web hosting provider
- Ensure the `index.html` file is in the root directory

### Updating Contact Information
1. **Phone Numbers**: Update in all HTML files and CSS

1. Add images to the `images/` folder
2. Update the CSS background-image properties
